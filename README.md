# Smart Business Messaging Platform

Context-aware ecommerce engagement platform that delivers personalized messaging across WhatsApp, SMS, Email, and Instagram.

## Features

- **Context-Aware Intelligence** — Leverages user browsing, purchase, and cart data for hyper-relevant messaging
- **Cross-Platform Messaging** — Unified experience across WhatsApp, SMS, Email, Instagram
- **Consent Management** — Ethical data handling with per-channel opt-in/opt-out
- **Smart Triggers** — Cart abandonment, browse nudges, post-purchase, win-back, welcome messages
- **Analytics Dashboard** — Channel performance, message tracking, engagement metrics
- **Campaign Management** — Create and manage targeted campaigns by segment

## Quick Start

```bash
npm install
npm start
```

Open `http://localhost:3000` in your browser.

## Tech Stack

- Node.js + Express (backend API)
- Vanilla JS (frontend dashboard)
- In-memory data store (demo-ready, no DB setup needed)
